# What is Python?

Python is a high-level programming language created to make coding simple and readable.

It allows programmers to write instructions that computers can understand and execute.

Python is widely used by beginners and professionals because of its clean syntax and versatility.

# Why Python

# Why Python?

Python is beginner-friendly and easy to understand compared to many other programming languages.

Its simple syntax helps new programmers focus on learning programming concepts without getting overwhelmed by complicated rules.

# Your First Python Program

Let's start with the most famous beginner program:

```python
print("Hello World!")
```

## Output

```text
Hello World!
```

---

# Understanding the Code

```python
print("Hello World!")
```

### Breakdown

- `print()` is a built-in Python function.
- It displays output on the screen.

- `"Hello World!"` is the text we want to display.
- Text inside quotes is called a **string**.

---

# Writing Comments

Comments are notes written inside your code.

Python *ignores comments* when running the program.

Comments help explain code and make it *easier to understand.*

## Example

```python
# This prints a message to the screen
print("Hello World!")
```

In Python, Anything after `#` (The Hashtag symbol) on the same line becomes a comment.

---

# Printing Multiple Lines

You can use multiple `print()` statements.

## Example

```python
print("Welcome to Python!")
print("This is Day 1.")
print("Let's start coding!")
```

## Output

```text
Welcome to Python!
This is Day 1.
Let's start coding!
```

# Common Beginner Mistakes

## Forgetting Quotes

Wrong:
```python
print(Hello)
```

Correct:
```python
print("Hello")
```

---

## Forgetting Parentheses

Wrong:
```python
print "Hello"
```

Correct:
```python
print("Hello")
```
